# SOCKS5 代理支持

## 概述

系统现在支持 SOCKS5 代理格式的订阅源。

## 支持的格式

### SOCKS5 文本格式
```
国家代码|IP地址:端口:用户名:密码
```

### 示例
```
CN|8.210.53.168:50635:ffH+M%12@Z:rks)M7-c09
US|38.209.120.60:43211:abc@1230123:abc@1230123
HK|45.192.98.205:51395:QRw2Sp*1!d:VVD4m2l773
```

## 使用方法

1. 在配置页面添加 SOCKS5 订阅源 URL
   - 例如: `https://backend.aihack.top/clash/socks5_list.txt`

2. 系统会自动识别并解析 SOCKS5 格式

3. 解析后的节点会转换为标准格式:
   ```
   socks5://用户名:密码@IP:端口#国家代码
   ```

## 功能特性

- ✅ 自动识别 SOCKS5 格式
- ✅ 支持特殊字符的用户名和密码（自动 URL 编码）
- ✅ 保留国家代码信息
- ✅ 支持 TCP 连通性测试
- ✅ 支持导出为 Clash YAML 格式
- ✅ 支持导出为 Base64 订阅格式
- ✅ 节点名称自动去重

## 节点标签格式

SOCKS5 节点的标签格式为:
```
🇨🇳|SOCKS5|IP:端口
```

例如:
- `🇨🇳|SOCKS5|8.210.53.168:50635`
- `🇺🇸|SOCKS5|38.209.120.60:43211`
- `🇭🇰|SOCKS5|45.192.98.205:51395`

## Clash 配置

导出的 Clash YAML 配置中，SOCKS5 节点格式为:
```yaml
- name: 节点名称
  type: socks5
  server: IP地址
  port: 端口
  username: 用户名
  password: 密码
```

## 注意事项

1. SOCKS5 格式必须严格遵循 `国家代码|IP:端口:用户名:密码` 的格式
2. 用户名和密码中的特殊字符会自动进行 URL 编码
3. 系统会对所有 SOCKS5 节点进行 TCP 连通性测试
4. 不可用的节点会被自动过滤
